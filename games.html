ForestQuest

June 6th, 2018

A very simple game that I made while learning programming fundamentals such as parameterization, conditionals, loops, and functions. One of the major takeaways I got from this project was using the parallax effect to create an illusion of character movement, even though the background is actually the thing that is moving.

CONTROLS: Move left [LEFT ARROW], move right [RIGHT ARROW], shoot orb [COMMAND]

Bouncing Cube

June 6th, 2018

A basic arcade game with multiple difficulties. Try to prevent the blocks from touching the bottom of the screen!

CONTROLS: Move left [LEFT ARROW], move right [RIGHT ARROW], start over [SHIFT], quit [ESCAPE]
